Search Catalogue

Using this window you query the named Cone service for any information it has about a region on the sky. If successful a VOTable catalogue will be returned that can have markers displayed on the current image.

Defining the search region

You can define the sky region using either an object name lookup, entering the RA and Dec (FK5/J2000) coordinates, making it the same region as the displayed image (Set area from image) or by dragging out a region of the displayed image (Select area...). The Size is in arcminutes.

Once you have defined the region to check press the Query button. That will initiate a query and if the service has any information rows will appear in the table.

Drawing markers

By default circles will be drawn on the image, you can change this using the Markers menu, or use more sophisticated marker that scale with magnitude for instance, using the Options->Set plot symbols... toolbox.
