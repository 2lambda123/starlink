Query VO Registry for TAP services

Using this window you can find any TAP (Table Access Protocol) services that have been registered in the VO.

You can use the old style R1.0 registries (from Astrogrid or the NVO) or the newer RegTap based ones (from GAVO at the time of writing). RegTap allows more flexible queries of the resources held by a registry, so is the default. In fact RegTap uses TAP itself, so is a more natural fit for queries.

To select a subset of services, add some qualifying substring to select amongst the various titles and press Query, you can then make TAP queries against the tables held by that service using by opening a query window by either selecting a row and pressing Open or by double clicking on a row.

Currently RegTap queries do not make use of the Column: attribute, so changing that will not have any effect. The registries column searched for the substring are: short_name, res_title, res_subjects, ivoid and res_description.

Once you have selected a suitable service. either double click on its name or select it and press Open that will open a further window in which you will enter the TAP query, but before you do that you may want to read the query window help.
