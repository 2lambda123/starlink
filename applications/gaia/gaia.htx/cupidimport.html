Import CUPID catalogue

This dialog helps you import a catalogue created by the CUPID (SUN/255) package.

To import a catalogue just select it's name and press Import. By default this will open the catalogue and display the STC-S shapes that it describes over the image. The supported STC-S shapes are ellipses and polygons.

To not use the STC-S shapes unselect STC Shape and choose which columns to use as RA and Dec. This will generally display rectangles based on the extents of the related columns.
