Configuring startup options

This window allows you to set the various options that GAIA supports when starting up. These will be saved when applied and then used the next time that GAIA starts. All these options can also be specified on the command-line, or by using X properties.

  • Use a floating panel (-float_panel)
    Create the control panel area in a detached window.
  • Display a zoom window (-with_zoom_window)
    Display, or not, the zoom window in the control panel.
  • Display a panner window (-with_pan_window)
    Display, or not, the panner window in the control panel.
  • Display a color ramp (-with_colorramp)
    Display, or not, the color ramp shown at the bottom of the main window.
  • Focus follows mouse (-focus_follows_mouse)
    Image and entry focus follows mouse pointer, alternative is to click on these first.
  • Show image scrollbars (-scrollbars)
    Display, or not, the scrollbars around the main image.
  • Make toolboxes transient (-transient_tools)
    Ask the window manager to make any toolboxes transient, or not. Transient windows may remain permanently above their parent window (depends on window manager properties).
  • Exit without prompt (-quiet_exit)
    When closing GAIA using the Exit item ask for confirmation before exiting, or not. Note that it is still possible to close GAIA without a request, if this is the last window and you use Close, or the window manager closes the last window.
  • Display milli-arcsecs (-extended_precision)
    The default precision for RA is three decimal places in seconds of time, for DEC two decimal places. If this option is selected an extra decimal place is added to give roughly milli-arcsecond resolution.
  • Minimum zoom (-min_zoom)
    The lowest zoom factor to be offered in the Scale: menu. Negative values are lower factors.
  • Maximum zoom (_max_zoom)
    The highest zoom factor to be offered in the Scale: menu. Note that the Scale: menu may run off the edges of the display if the difference between the minimum zoom and this value is too large.
  • Default colormap (-default_cmap)
    The colormap to be used by default. Note that this isn't as useful as it might seem as images previously viewed will be shown with the colormap they where last displayed with, and the default colormap becomes the one last selected.
  • Default itt (-default_itt)
    The default intensity transfer table (the notes about colormaps also apply to this option).
