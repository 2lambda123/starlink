Starting the Software

Portable-CGS4DR can be started with three optional command line parameters:

$1 is a data directory /${home}/
$2 is a UT date /current UT-date/
$3 is the file type DST or NDF /ndf/

To start the software, source the Starlink login and cshrc files and use your local data directory. E.g.:

% source /star/etc/login
% source /star/etc/cshrc
% cgs4dr /scratch/pnd/19940811 940811 ndf
Welcome to Portable-CGS4DR V1.3-0
ICL>

There is also a `setup' facility for defining environmental variables without starting the software:

% source /star/etc/login
% source /star/etc/cshrc
% cgs4dr_setup /scratch/pnd/19940811 940811 ndf
Setup Portable-CGS4DR V1.3-0 for /scratch/pnd/19940811
%
