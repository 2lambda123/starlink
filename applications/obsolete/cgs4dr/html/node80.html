enter_obs_range

ICL> enter_obs_range p1 p2 p3
Use: Recursively writes ``REDUCE oyymmdd_n'' and ``END oyymmdd_n'' pairs (for n = p1 to p2) to database for pre-defined UT-date
p1 = Start observation number /1/
p2 = End observation number /p1/
p3 = Queue position /`oldest'/
E.g: enter_obs_range 1 5 oldest
