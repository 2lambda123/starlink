Version History

This is the fourth incantation of the Portable-CGS4DR software. The functionality of each release is:

V1.0-0
  1. First release of Portable-CGS4DR.
  2. Supports reduction of all frames into calibrated or uncalibrated reduced observations and groups in DST or NDF format using STARE or NDSTARE observing modes.
  3. Supports calibration of ARC into reduced calibration frames.
  4. Supports observation filing, dividing by a standard source and automated extraction of spectra.
  5. Supports ICL command line interface.
V1.1-0
  1. Supports all features in V1.0-0.
  2. Supports user defined bad pixel masks.
  3. Supports 256x256 ALICE data in all modes (including CHOP).
  4. New features include set_/get_polyfit, set_/get_file_selection, set_/get_cut_dir and oversampled FLAT fields.
  5. Bug fixes for endgroup and plot_graph.
  6. Support for a demo.
V1.2-0
  1. Supports all features in V1.1-0.
  2. Bug fixes for pause_on_error, ukirtfig and NBS.
  3. Supports TCLADAM GUI.
V1.3-0
  1. Supports all features in V1.2-0.
  2. Corrects problems with bad pixel mask generation.
  3. Bug fixes to irflat, deripple, create_window_mask, divide_by_ff.
  4. Conforms to Figaro V5 structure.
  5. Removes NAG dependency by using PDA_ routines.
  6. Essential engineering functions added (two row/col line fitting, cursor_pixel and array_tests).
  7. Re-design of some GUIs saving 35% in real estate.
  8. Supports dynamic defaults in GUI.
  9. Supports on-line help in GUI.
  10. Fully documented on WWW (using client-side image maps).
  11. Adds default foreground/background colours to GUI.
  12. Now the default version for DR at UKIRT (using distributed ADAMnet).
  13. Enabled print of textpane and gwm to print device.
