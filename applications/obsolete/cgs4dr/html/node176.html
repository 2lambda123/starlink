How Do I List the Available Masks and Colour Tables?

Under Portable-CGS4DR both masks and colour tables are held in the same sub-directory. Actually there are two sub-directories, one for DST data and one for NDF data. The commands for listing NDF data and DST data from the shell are respectively:

% ls $CGS4DR_ROOT/ndf
% ls $CGS4DR_ROOT/dst

Alternatively, if you are using the graphical user interface, you can select the following:

  • List Masks from the control task options menu for a list of bad pixel and window masks.
  • Set Look-Up-Table from the plotting task options menu for a list(box) of available colour tables.
