Reducing a Dark

To reduce a DARK, Portable-CGS4DR reads the raw data from store in $IDIR/ and FITS header information from store in $ODIR/, applies a bad pixel mask, optionally subtracts a reduced BIAS frame if in stare mode, linearises the data (if required) and writes the reduced data parameters into the index file and stores the reduced DARK in $RODIR/ as shown in figure 2.1.

Note that DARK frames are not normally subtracted when the data is reduced as OBJECT-SKY pairs.
