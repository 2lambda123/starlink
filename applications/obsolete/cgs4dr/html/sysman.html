System Manager's Page

Portable-CGS4DR is distributed using the Starlink mk/makefile paradigm. The system administrator will usually get a compressed tar file which must be unpacked and built for the appropriate architechture:

% setenv SYSTEM sun4_Solaris
or
% setenv SYSTEM alpha_OSF1
followed by:
% setenv INSTALL /star/bin
% cd /star/cgs4dr
% zcat cgs4dr.tar.Z | tar xvf -
% ./mk check clean build strip install
The full Portable-CGS4DR release takes about 7 Mb of disk space. The demo is installed by default but is optional and a saving of 6 Mb can be made if the demo data is deleted:
% rm -rf $CGS4DR_ROOT/demo
