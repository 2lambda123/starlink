How Do I Re-Normalize a FLAT Field?

If you forget to normalize a FLAT field, the observation should be re-entered into the queue at some appropriate point. The FLAT field normalisation parameters should be altered using the Reduction Params action button from the control task interface.

Once the FLAT field has been successfully re-reduced, the reduction task monolith, RED4, must be reset to force the mapping of the new reduced observation file. To reset any task, see the section How Do I Reset the Tasks?.
