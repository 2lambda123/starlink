The Set Commands

P4 maintains a noticeboard for each port and these items are used tailor the display in a variety of ways. The more common items can be accessed through the set_itemn commands. The suite of commands available are shown in table 9.1. The set_item commands ( i.e. without a specific port number attached) are treated as a special case and assign the given value to all ports.
