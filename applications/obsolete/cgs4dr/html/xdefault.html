Changing X Defaults

Portable-CGS4DR now features customizable X defaults located in the file $CGS4DR_CONFIGS/cgs4dr.xopts. This is a simple ASCII text file and features can be removed by preceding any line with the comment character (#). New lines may be added using the same style contained within the file.

Upon startup, a default file is copied to $CGS4DR_CONFIGS/cgs4dr.xopts if none exists.
