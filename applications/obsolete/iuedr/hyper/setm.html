IUEDR-Online HELP

SETM

Summary

Set dataset parameters that are ORDER specific. 

   DATASET         Dataset name.
   ORDER           Echelle order number.
   RIPK            Echelle ripple constant (A).
   RIPA            Ripple function scale factor.
   RIPC            Ripple function correction polynomial.
   WCUT            Wavelength limits for Echelle order (A).

Purpose

This command allows changes to be made to dataset values which are specific to the specified ORDER . Items for which parameters are not specified retain their current values.

Available Topics

DATASET * ORDER * RIPK * RIPA * RIPC * WCUT
