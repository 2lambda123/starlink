XCUT

Usage

XCUT=number,number

Description

This is one of the mechanisms that can be used to delimit the ends of Echelle orders that are calibrated for ripple response. The two values of this parameter are the start and end "X" coordinates of the order, where: 

      X = PI * RIPA * (WAV - WAVC) * ORDER / WAVC

and PI=3.14159, WAVC=RIPK/ORDER. The nature of the standard ripple function is that X is only formally meaningful in the range (-PI, +PI).

See the RIPK , RIPC and RIPA parameter descriptions.
