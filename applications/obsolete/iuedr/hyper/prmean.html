PRMEAN

Summary

Print current mean spectrum in tabular form. 

   DATASET         Dataset name.

Purpose

This command prints the mean spectrum associated with DATASET in tabular form.

The table consists wavelengths and calibrated flux, along with any data quality information.

The output from this command should be diverted to a file, since it is likely to be to voluminous to read at the terminal.
