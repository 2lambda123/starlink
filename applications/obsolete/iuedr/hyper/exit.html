IUEDR-Online HELP

EXIT

Summary

Leaves IUEDR and returns to the command shell.

Purpose

This command leaves the IUEDR programme and returns to the shell from which it was started. Any data files which require update will be written by this command. The command is a synonym for the QUIT command.

Note that the existing files for the current dataset are overwritten.
