BARKER

Summary

The spectrum data in DATASET are corrected for echelle ripple in the range ORDERS using the method of Barker (1984). 

   DATASET         Dataset name.
   ORDERS          This delineates a range of Echelle orders.

Purpose

The spectrum data in DATASET are corrected for residual echelle ripple using the method described by Barker (1984. Astronomical Journal, Vol. 89, 899). Orders in the range ORDERS are used in the ripple correction optimisation. Note that this optimisation method is only applicable for SWP spectra.
