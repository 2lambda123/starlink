APERTURES

Usage

APERTURES=name

Description

This specifies an aperture or group of apertures. The following names have defined meanings: 

      SAP       IUE Small Aperture
      LAP       IUE Large Aperture
      BAP       IUE Both Apertures (i.e. SAP and LAP)
