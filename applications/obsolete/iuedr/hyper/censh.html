IUEDR-Online HELP

CENSH

Usage

CENSH=boolean

Description

This indicates whether the spectrum signal produces a single linear shift to the initial template.

This can be used in cases whether the object signal is too weak to provide a detailed centroid determination, but it can at least be used to move a pre-existing template shape into the right position.
