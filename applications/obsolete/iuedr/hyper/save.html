IUEDR-Online HELP

SAVE

Summary

Write new versions for any files that have had their contents changed in the current session.

Purpose

This command forces new versions to be written for dataset files that have had their contents changed during the current session. If there are no outstanding files then this command does nothing.

Note that the existing files for the current dataset are overwritten.
