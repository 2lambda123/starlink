IUEDR-Online HELP

FIDFILE

Usage

FIDFILE=string

Description

This is the name of a file containing fiducial positions. A type of ".fid" is assumed, and should not be specified explicitly.

An environment variable may be used or pre-pended to the filename, for example to load the supplied fiducial file for a SWP camera PHOT image: 

   > NEWFID FIDFILE=$IUEDR_DATA/swpr
