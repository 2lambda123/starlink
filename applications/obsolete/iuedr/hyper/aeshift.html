AESHIFT

Summary

Determine echelle shift by measuring shifts for several absorption line features in a HIRES image. 

   DATASET         Dataset name.
   CENTREWAVE      Line central wavelengths (A).
   DELTAWAVE       Half-width of line search windows (A).

Purpose

AESHIFT can be used to measure the global echelle shift for a HIRES spectrum. A set of laboratory wavelengths of absorption features which should be present in the spectrum are located in the spectrum and the ESHIFT for each of them is found. The median value of these ESHIFT s is then selected and used for the whole dataset.
