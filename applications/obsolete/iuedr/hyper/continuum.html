CONTINUUM

Usage

CONTINUUM=boolean

Description

This indicates whether the object spectrum is expected to contain a significant continuum. It is used in conjunction with the EXTENDED parameter in determining the positions and widths of object and background channels for spectrum extraction from HIRES datasets.
