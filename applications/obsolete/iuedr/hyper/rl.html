IUEDR-Online HELP

RL

Usage

RL=number,number

Description

This is a pair of radial coordinate values defining the start and end of the radial grid in an LBLS array. These radial coordinates are measured in geometric pixels, and run perpendicular to the dispersion direction. A coordinate of 0.0 corresponds to the centre of the spectrum.

Values of [0.0,0.0] indicate that internal defaults are to be adopted. A single value is reflected symmetrically about 0.0.
