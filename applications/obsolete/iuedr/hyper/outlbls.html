IUEDR-Online HELP

OUTLBLS

Summary

Output the current LBLS array to a binary sequential file. 

   DATASET         Dataset name.
   OUTFILE         Name of output file.

Purpose

This command outputs the current LBLS array to a file. The file name, if not specified by the OUTFILE parameter, is constructed: 

      <CAMERA><IMAGE>R.DAT

The format of this file is described by the Fortran 77 routine, RDLBLS, which can be found in the file: 

      $IUEDR_USER/rdlbls.for

this directory (environment variable $IUEDR_USER) also contains a test program for using RDLBLS, and other helpful items.
