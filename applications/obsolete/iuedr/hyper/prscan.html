IUEDR-Online HELP

PRSCAN

Summary

Tabulate intensities of current scan. 

   DATASET         Dataset name.

Purpose

This command prints the scan associated with DATASET in tabular form. The table consists wavelengths and net, along with any data quality information.

The output from this command should be diverted to a file, since it is likely to be to voluminous to read at the terminal.
