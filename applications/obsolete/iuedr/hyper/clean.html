CLEAN

Summary

This command is used to mark pixels with values below a selected threshold as BAD. 

   DATASET         Dataset name.
   THRESH          Smallest pixel value to be accepted as GOOD.

Purpose

Some IUE datasets are effected by horizontal bars of low pixel values. These are caused by a weak signal from the IUE craft at the time of data download. When there are a few bars they can be marked as bad with the EDIMAGE command. In the case of many bars a quicker solution is to mark all pixels in the image below a user selected threshold value as BAD.

Successive CLEAN and DRIMAGE commands starting with a value of THRESH as -1000 and increasing THRESH towards zero will allow the user to chose a value suitable for the problem image.
