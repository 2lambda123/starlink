IUEDR-Online HELP

LISTIUE

Summary

Analyse the contents of one or more IUE tape files. 

   DRIVE           Tape Drive or File Name.
   FILE            File Number.
   NFILE           Number of Tape Files to be processed.
   NLINE           Number of IUE header lines printed.
   SKIPNEXT        Whether skip to next file.

Purpose

This performs an analysis of NFILE IUE tape files, starting at file specified by the FILE parameter. NFILE =-1 means all files until the end of the tape. This can also be used to display the header data from a GO format disk file.

Available Topics

DRIVEFILE * FILE * NFILE * NLINE * SKIPNEXT
