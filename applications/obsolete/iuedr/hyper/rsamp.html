IUEDR-Online HELP

RSAMP

Usage

RSAMP=number

Description

This is the sample spacing used for the radial grid in the LBLS array. If it does not fit an integral number of times into the grid limits, RL , then the latter are adjusted to fit.

Suggested values range from 0.707 to 1.414 pixels; the latter corresponding to the IUESIPS LBLS grid.
