IUEDR-Online HELP

APERTURE

Usage

APERTURE=name

Description

This is the name of an individual aperture. The following names have defined meanings: 

      SAP       IUE Small Aperture
      LAP       IUE Large Aperture

Other apertures names may also be defined.
