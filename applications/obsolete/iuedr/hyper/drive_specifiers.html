IUEDR-Online HELP

Drive specifiers

Summary

The use of the DRIVE parameter has been enhanced to allow specification of disk files containing IUE datasets. This is intended for use with files obtained from online archives (RAL and NASA).

The syntax is to provide the full filename and extension in response to the DRIVE prompt.

E.g. 

      DRIVE> SWP12345.RAW
