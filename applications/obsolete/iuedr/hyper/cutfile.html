IUEDR-Online HELP

CUTFILE

Usage

CUTFILE=string

Description

This is the name of a file containing Echelle order wavelength limits. A file type of ".cut" is assumed, and should not be specified explicitly.

An environment variable may be used or pre-pended to the filename, for example to load the supplied cut file for the SWP camera LAP: 

   > NEWCUT CUTFILE=$IUEDR_DATA/swplap
