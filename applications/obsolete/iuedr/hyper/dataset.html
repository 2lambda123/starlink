IUEDR-Online HELP

DATASET

Usage

DATASET=filename

Description

This is the root name of the file containing the dataset. The file type (e.g. "_UED.sdf") should not be given in the DATASET name. If the file name contains a directory specification, it should be enclosed in quotes.
