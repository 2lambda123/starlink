IUEDR-Online HELP

YP

Usage

YP=number,number

Description

This specifies the pixel limits along the y-direction used for image display. Values in decreasing order will cause the image to be inverted along the y-direction. If the values are undefined, the pixel limits will default to include the whole extent of the image along the y-direction.
