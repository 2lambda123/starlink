IUEDR-Online HELP

THDA

Usage

THDA=number

Description

This is the IUE Camera Temperature, measured in degrees C. It is used for such things as adjustments to Fiducial positions and Spectrograph Dispersion Relations. A value of 0.0 implies that no value is available. A suitable value can be found in the IUE Header Text of the final spectrum file on the Guest Observer Tape (IUESIPS 2 only).
