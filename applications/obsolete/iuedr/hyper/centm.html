IUEDR-Online HELP

CENTM

Usage

CENTM=boolean

Description

This indicates whether a centroid template from the calibration dataset is used as a start in defining the precise position of the spectrum signal on the image.

There is no guarantee that such a template will be available in a particular situation (camera/aperture/resolution/order).
