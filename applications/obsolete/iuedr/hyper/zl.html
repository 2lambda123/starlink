IUEDR-Online HELP

ZL

Usage

ZL=number,number

Description

This specifies the data limits used for display of images. If the values are given in decreasing order, then high data values will be represented by low (dark) display intensities, and vice-versa. If the values are undefined, then the full intensity range of the image will be used.

Data values which fall below the lowest display intensity are drawn in BLACK, while those which fall above the highest display intensity are drawn BLUE.
