IUEDR-Online HELP

DISPFILE

Usage

DISPFILE=string

Description

This is the name of a file containing dispersion data. A type of ".dsp" is assumed, and should not be specified explicitly.

An environment variable may be used or pre-pended to the filename, for example to load the supplied dispersion file for the SWP camera HIRES mode: 

   > NEWDISP DISPFILE=$IUEDR_DATA/swphi
