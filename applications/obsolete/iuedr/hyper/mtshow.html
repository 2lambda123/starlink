IUEDR-Online HELP

MTSHOW

Summary

Show tape position. 

   DRIVE           Tape Drive Name.

Purpose

This command displays the current tape position. This includes the file number, and the block position, either relative to the start or the end of the file.

Note that the actual file position may differ from the value of the FILE parameter.
