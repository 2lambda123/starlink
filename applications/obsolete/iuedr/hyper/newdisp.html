NEWDISP

Summary

Read spectrograph dispersion data into DATASET from text file with name specified by DISPFILE . 

   DATASET         Dataset name.
   DISPFILE        Name of file containing dispersion data.

Purpose

This command reads spectrograph dispersion data from a text file specified by the DISPFILE parameter, and stores them in the dataset specified by DATASET .

Available Topics

DATASET * DISPFILE
