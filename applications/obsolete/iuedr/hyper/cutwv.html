IUEDR-Online HELP

CUTWV

Usage

CUTWV=boolean

Description

This indicates whether any available Echelle order wavelength cutoff limits are to be used for the spectrum extraction wavelength grid limits. Highly recommended, provided that you are happy with these wavelength limits.
