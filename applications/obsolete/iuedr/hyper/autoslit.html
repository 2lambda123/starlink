IUEDR-Online HELP

AUTOSLIT

Usage

AUTOSLIT=boolean

Description

This determines whether the extraction slit is determined automatically by the command. When AUTOSLIT=TRUE the BDIST , BSLIT and GSLIT parameter values are determined automatically, based on the IUE Camera, Resolution, Aperture, and on the values of the CONTINUUM and EXTENDED parameters. This mode of operation is probably the best for point source objects.
