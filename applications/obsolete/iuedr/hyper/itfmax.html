ITFMAX

Usage

ITFMAX=number

Description

This is the pixel value on tape corresponding to ITF saturation. Its value is fixed for a given ITF table. The value of ITFMAX is only needed for IUE images of type "GPHOT". The appropriate value can be determined from inspection of the IUE Text Header for the GPHOT file, and in particular the table of numbers following the line which begins "PCF C/**". Here are the ITFMAX values associated with various forms of this table: 

      ITFMAX    TABLE                           IDENTIFICATION
      20000     0  1800  3700  5600  ... 30000  Preliminary LWR ITF
      27220     0  2303  4069  8008  ... 42032  2nd LWR ITF
      19983     0  1800  3600  5500  ...        Preliminary SWP ITF
      19740     0  1753  3461  6936  ... 28674  Faulty, 2nd SWP ITF
      19632     0  1684  3374  6873  ... 28500  Corrected, 3rd SWP ITF
