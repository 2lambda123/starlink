UNIX to VAX data conversion

Summary

If you wish to transfer IUE image data from a UNIX machine onto a VAX in order to use the VAX IUEDR then the transfer should be done using FTP (in BINARY mode) and the command 

      VAX_FORMAT  image-name

should then be used to ensure the file has the correct format.

If you use some other method of transferring the file (eg NFS) then the above command is STILL required.
