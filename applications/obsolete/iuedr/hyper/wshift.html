WSHIFT

Usage

WSHIFT=number

Description

This is a constant wavelength shift applied to spectrum wavelengths. It is measured in (A), and affects the spectrum wavelengths: 

      WAV(NEW) = WAV(OLD) + WSHIFT

This is only used for LORES spectra.
