IUEDR-Online HELP

PRSPEC

Summary

Print current aperture/order spectrum in tabular form. 

   DATASET         Dataset name.
   APERTURE        Aperture name (SAP or LAP).
   ORDER           Echelle order number.

Purpose

This command prints the spectrum associated with ORDER or APERTURE and DATASET in tabular form.

The table consists wavelengths, net and calibrated fluxes, along with any data quality information.

The output from this command should be diverted to a file, since it is likely to be to voluminous to read at the terminal.

Available Topics

DATASET * APERTURE * ORDER
