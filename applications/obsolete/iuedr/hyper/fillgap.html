IUEDR-Online HELP

FILLGAP

Usage

FILLGAP=boolean

Description

If a grid point in the mean spectrum would have had a contribution from a bad data point, this parameter determines whether that grid point is marked as unusable within the context of the order or aperture being mapped. If the grid point is marked as unusable in this way then other (good) points cannot contribute to it.
