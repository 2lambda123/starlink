IUEDR-Online HELP

DEVICE

Usage

DEVICE=name

Description

This is the GKS/SGS graphics device. A list of possible GKS workstations may be found in SUN/83 . A list of SGS workstation names available at your site may be obtained either by a null response to the DEVICE parameter prompt (i.e. "!"), or by using the IUEDR command SGS .
