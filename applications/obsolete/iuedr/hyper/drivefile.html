IUEDR-Online HELP

DRIVE

Usage

DRIVE=name

Description

This the device name of the Tape Drive. Feasible values are /dev/nrmt0h on a unix machine or MSA0 on a VMS machine. Other values may apply, check with your system manager.

Alternatively, a filename may be given. If the filename contains a directory specification it should be placed in single quotes.
