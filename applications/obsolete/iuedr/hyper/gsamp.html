IUEDR-Online HELP

GSAMP

Usage

GSAMP=number

Description

This is the sampling rate used for spectrum extraction. It is measured in "geometric pixels". GSAMP=1.414 corresponds to the original IUESIPS (1) sampling rate, while GSAMP=0.707 corresponds to the later IUESIPS (2) sampling rate. Other values can be chosen.
