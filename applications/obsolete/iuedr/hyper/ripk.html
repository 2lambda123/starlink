RIPK

Usage

RIPK=number

Description

This is the Echelle ripple constant measured in (A). It corresponds to the central wavelength of Echelle order number 1. The central wavelength of an arbitrary ORDER is: 

      WAVC = RIPK / ORDER

Where this parameter is used for an entire HIRES dataset, the parameter can have more than 1 value, and represent a polynomial in ORDER: 

      WAVC = (RIPK(1) + RIPK(2) * ORDER + RIPK(3) * ORDER**2...) / ORDER

See the RIPA and RIPC parameter descriptions.
