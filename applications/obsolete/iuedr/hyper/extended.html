IUEDR-Online HELP

EXTENDED

Usage

EXTENDED=boolean

Description

This indicates whether the object spectrum is expected to be extended (rather than a point source). It is used in conjunction with the CONTINUUM parameter in determining the positions and widths of the object and background channels used for spectrum extraction from HIRES datasets.
