IUEDR-Online HELP

RIPFILE

Usage

RIPFILE=string

Description

This is the name of a file containing an Echelle ripple calibration. A type of ".rip" is assumed, and should not be specified explicitly.

An environment variable may be used or pre-pended to the filename, for example to load the supplied ripple file for the SWP camera: 

   > NEWRIP RIPFILE=$IUEDR_DATA/swp
