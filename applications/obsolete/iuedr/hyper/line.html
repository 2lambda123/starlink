LINE

Usage

LINE=name

Description

This specifies the line style to be used for the next curve to be plotted. It can be one of the following: 

      SOLID        Solid (continuous) line
      DASH         Dashed line
      DOTDASH      Dot-dash line
      DOT          Dotted line

The order of these is that invoked when automatic line style rotation is in effect (see the LINEROT parameter).
