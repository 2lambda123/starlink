COLROT

Usage

COLROT=boolean

Description

This indicates whether the line colour is to be changed after the next plot. The initial line has colour index 1 SOLID, unless specified explicitly using the COL parameter. The sequence of colour indices goes (1, 2, 3, ..., 10, 1, 2, ...).

In commands where more than one line is plotted, COLROT determines whether these lines have different colours.

Lines will only appear with different colours on displays which support this!, on other devices COLROT is harmless.
