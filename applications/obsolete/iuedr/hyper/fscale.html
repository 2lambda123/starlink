FSCALE

Usage

FSCALE=number

Description

This is an (arbitrary) scale factor applied to spectrum fluxes. It affects fluxes: 

      FLUX(NEW) = FLUX(OLD) * FSCALE

It finds application in accounting for grey attenuation, or obscuration of object signal through a narrow aperture.
