RIPA

Usage

RIPA=number

Description

This is an empirical scale factor that can be used to modify the Echelle ripple function. The normal value is 1. The primary component of the ripple function is: 

      SCALE = (SIN(X) / X)**2

where 

      X = PI * RIPA * (WAV - WAVC) * ORDER / WAVC

and PI=3.14159, WAVC=RIPK/ORDER. The net spectrum is divided by SCALE above. Empirical values of RIPA can be used to optimise the ripple correction.

See the RIPC and RIPK parameter descriptions.
