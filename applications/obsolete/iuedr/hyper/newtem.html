Summary

Read spectrum centroid template data into DATASET from text file with name specified by TEMFILE . 

   DATASET         Dataset name.
   TEMFILE         Name of file containing spectrum template data.

Purpose

Read spectrum centroid template data into DATASET from text file with name specified by TEMFILE .

The file type is assumed to be ".tem", and should not be specified as part of the TEMFILE parameter.
