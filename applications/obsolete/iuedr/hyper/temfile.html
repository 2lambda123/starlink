IUEDR-Online HELP

TEMFILE

Usage

TEMFILE=string

Description

This is the name of a file which contains the standard spectrum template data. A type of ".tem" is assumed, which should not be specified explicitly.

An environment variable may be used or pre-pended to the filename, for example to load the a template file "swptemp" from the a directory named IUEDR_TEM: 

   > NEWTEM TEMFILE=$IUEDR_TEM/swptemp
