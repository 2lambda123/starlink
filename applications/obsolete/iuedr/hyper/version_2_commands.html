IUEDR Version 2 Commands

Summary

Several commands have been superceded in the current version of IUEDR . The following table gives a list of the new commands. 

    Old Command  New Command       Old Command  New Command


    ABSCAL       NEWABS            PF           PLFLUX
    CU           CURSOR            PM           PLMEAN
    CUTCAL       NEWCUT            PN           PLNET
    DI           DRIMAGE           PRINTGRS     PRGRS
    DISPCAL      NEWDISP           PRINTLBLS    PRLBLS
    EI           EDIMAGE           PRINTMEAN    PRMEAN
    EM           EDMEAN            PRINTSCAN    PRSCAN
    ES           EDSPEC            PRINTSPEC    PRSPEC
    FIDCAL       NEWFID            PS           PLSCAN
    MI           MODIMAGE          READIUERS    READSIPS
    PC           PLCEN             RIPCAL       NEWRIP
    PE           PLGRS             TEMCAL       NEWTEM

ABSCAL

Read absolute flux calibration into DATASET . This command has been replaced by the NEWABS command.

CU

Find display coordinate using the cursor and print on terminal. This command has been replaced by the CURSOR command.

CUTCAL

Read echelle order wavelength limits into DATASET . This command has been replaced by the NEWCUT command.

DI

Display image on an image display device. This command has been replaced by the DRIMAGE command.

DISPCAL

Read spectrograph dispersion data into DATASET . This command has been replaced by the NEWDISP command.

EI

Edit image data quality using image display cursor. This command has been replaced by the EDIMAGE command.

EM

Edit mean spectrum using graphics cursor. This command has been replaced by the EDMEAN command.

ES

Edit net spectrum using graphics cursor. This command has been replaced by the EDSPEC command.

FIDCAL

Read Fiducial positions into DATASET . This command has been replaced by the NEWFID command.

MI

Modifies actual image intensities interactively. This command has been replaced by the MODIMAGE command.

PC

Plot smoothed centroid shifts. This command has been replaced by the PLCEN command.

PE

Plot pseudo-gross and background from spectrum extraction. This command has been replaced by the PLGRS command.

PF

Plot calibrated flux spectrum. This command has been replaced by the PLFLUX command.

PM

Plot mean spectrum. This command has been replaced by the PLMEAN command.

PN

Plot uncalibrated net spectrum. this command has been replaced by the PLNET command.

PRINTGRS

Print recently extracted aperture/order spectrum in tabular form. This command has been replaced by the PRGRS command.

PRINTLBLS

Print current LBLS array in tabular form. This command has been replaced by the PRLBLS command.

PRINTMEAN

Print current mean spectrum in tabular form. This command has been replaced by the PRMEAN command.

PRINTSCAN

Tabulate intensities of current scan. This command has been replaced by the PRSCAN command.

PRINTSPEC

Print current aperture/order spectrum in tabular form. This command has been replaced by the PRSPEC command.

PS

Plot scan perpendicular to dispersion. This command has been replaced by the PLSCAN command.

READIUERS

Reads MELO/MEHI from IUESIPS 1 or IUESIPS 2 tape. This command has been replaced by the READSIPS command.

RIPCAL

Read echelle ripple calibration into DATASET . This command has been replaced by the NEWRIP command.

TEMCAL

Read spectrum centroid template data into DATASET . This command has been replaced by the NEWTEM command.
