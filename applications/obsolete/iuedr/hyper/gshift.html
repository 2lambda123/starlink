GSHIFT

Usage

GSHIFT=number,number

Description

This is a global constant shift of the spectrum format on the image, (DX, DY), where the geometric coordinates, (X, Y) of a spectrum position are: 

      X(NEW) = X(OLD) + DX
      Y(NEW) = Y(OLD) + DY
