IUEDR-Online HELP

FLAG

Usage

FLAG=boolean

Description

This specifies whether the data quality information is displayed along with the image. If so, then faulty pixels will be marked with a colour according to the following scheme: 

      GREEN       pixels affected by reseau marks
      RED         pixels which are saturated (DN=255)
      ORANGE      pixels affected by ITF truncation
      YELLOW      pixels marked bad by the user

User edits are only shown where no other fault is present.

This option would normally only be used when assessing the quality of faulty pixels, possible with a view to using them (marking them "good" with a cursor editor).
