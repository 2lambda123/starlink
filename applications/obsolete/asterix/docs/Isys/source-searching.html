Point Source Searching

Selecting Search loaded image under the Sources item in the main menu will invoke a GUI to the Asterix point source searching application, PSS, as shown on the left. Click on the live parts of this screen shot to view more detail.

Note that although Isys will try to run PSS automatically, this may not always work and PSS may ask for further parameter values. You should preferably be familiar with PSS to be able to respond to this. The main area this may happen is with the definition of the point spread function.
