Formatting Plots


This group of buttons call up dialogues which allow various attributes of the plot to be set, or textual and graphical additions to be applied.

  • set default plotting attributes
  • set plotting window position
  • set various axis attributes
  • put titles above plot
  • set axis labels
  • put annotations on plot
  • overlay coordinate plot
  • put key at side of plot
  • set colour table


To preserve plot formatting the image or 1D plot must be saved. The same formats can then be loaded into another session.
