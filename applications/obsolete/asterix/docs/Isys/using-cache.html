Using Cache



Whenever a change is made to the image, Isys keeps a copy of the unchanged image in a private cache. This button will bring back the unchanged image to the current image buffer. This is the only interaction the user has with this private cache, ie. it lets you undo one step of processing.
This button makes a copy of the current image in a public cache which the user then has access to via the other buttons.
Replaces the current image with the cached one.
Swaps the current and cached images.
Deletes the cached image.
Adds the cached image to the current one.
Subtracts the cached image from the current one.
