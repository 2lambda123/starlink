Point Source Searching


Input Name of image file tobe searched. By default this is the image currently loaded into Isys. PSS will read the original file and will not therefore see any changes that Isys has made.
Background This can be a single background level, or a background image file.
Search Region The region of the image to be searched. By default this is a box enclosing the current region.
Significance Map Optional output significance map.
Source subtracted image Optional output image with sources removed.
Output source list Name of output source list. This list is fed back to Isys.
