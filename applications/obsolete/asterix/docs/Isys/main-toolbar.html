Main Toolbar


Image Forces the loaded image to be replotted.
1D-Plot Forces 1D data to be replotted.
Scaling Sets the scaling algorithm by which the data are mapped to the colour table. This is set via a dialogue.
Contours Brings up a dialogue to allow contours to be overlayed on the image.
Smooth Allows the data to be smoothed. A simple dialogue allows the degree of smoothing to be set.
Browse Tracks the cursor position and displays data values around that position in a floating window. There are buttons in that window to switch the display to variance, error, significance or quality.
SetPos Set the current position. When this button is pressed, Isys tracks the cursor and displays the position in the status box. Clicking the left button will set the position. Clicking the right button will open the position dialogue. Note that even without pressing this button, clicking the middle button in the image will always set the position.
Centroid Finds the centroid of the given circle and makes it the current position.
