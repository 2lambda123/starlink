Main Menu



File Loading and saving files. Image files, 1D data and spatial description files and loaded and saved via this menu item.
Edit Invoke a number of operations which make changes to the image. The operations that can be invoked are: smoothing, cut-and-paste, adding noise.
Options User-selectable options.
View Controls which toolbars, etc are visible. The following can be toggled on and off: the message box, the plot formatting toolbar, the region toolbar, the cache toolbar.
Sources Invoke source searching and catalogue/archive searches.
Profiles Create 1D profiles. Available profiles are: radial, azimuthal, arbitrary rectangular slice.
Regions Simple region selection. For more comprehensive region handling use the region toolbar.
Mark Mark positions on image. You can mark the current position, the current list of positions, or remove current marks. See also the section on setting positions.
Zoom Set zooming factor. The plot can be zoomed in about the current position. Alternatively you can zoom in about the current region. Note that zooming only affects the display and doesn't change the current region.
