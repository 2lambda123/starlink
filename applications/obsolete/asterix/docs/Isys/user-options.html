User Options


Auto refresh display With this option checked the image will automatically be redrawn whenever a change is made to the data or the plot format
Auto save settings Will automatically record changes, such as which toolbars are visible, in the user's configuration file, so that they will be preserved for subsequent sessions.
Show tips on startup Show dialogue with helpfull hints when Isys starts up.
External 1D display Plot 1D data in a separate window, rather than sharing the internal plotting window with the image.
Resize display Set the size of the internal plotting window.
