IMSTAT -
Get statistics of an image

Calculates statistics of an area of the image

Takes an input area of an image, and calculates the mean and standard deviation, minimum and maximum pixel values (with their locations), the number of valid and invalid pixels, and the sum of the valid pixel values.

The results are typed out, and also put as parameters of the program

2.1 In

(Default - null)

The name of the input image file.

The input image must be a standard Starman ADAM .SDF image, with either `real' 32-bit real values, or `short' 16-bit integer values. It can have the `BSCALE' and `BZERO' values, and `magic pixel values' indicating `bad' pixels.

2.2 Xrange

(Default - 1,input image X size
Range - 1 to input image X size)

2.3 Yrange

(Default - 1,input image Y size
Range - 1 to input image Y size)

Y start and Y end of the area of image to analyse

2.4 Output parameters

2.4.1 Max

(output parameter)

Set by program. The maximum valid pixel value in the selected area.

2.4.2 Mean

(output parameter)

Set by program. The mean of the valid pixel values in the selected area.

2.4.3 Min

(output parameter)

Set by program. The minimum valid pixel value in the selected area.

2.4.4 Nbad

(output parameter)

Set by program. The number of invalid pixels in the selected area.

2.4.5 Ngood

(output parameter)

Set by program. The number of valid pixels in the selected area.

2.4.6 Stddev

(output parameter)

Set by program. The standard deviation of the valid pixel values in the selected area.

2.4.7 Sum

(output parameter)

Set by program. The sum of the values of the valid pixels in the selected area.

2.4.8 Xmaxloc

(output parameter)

Set by program. The X location of the pixel with the maximum value in the selected area.

2.4.9 Xminloc

(output parameter)

Set by program. The X location of the pixel with the minimum value in the selected area.

X start and X end of the area of image to analyse

2.4.10 Ymaxloc

(output parameter)

Set by program. The Y location of the pixel with the maximum value in the selected area.

2.4.11 Yminloc

(output parameter)

Set by program. The Y location of the pixel with the minimum value in the selected area.
