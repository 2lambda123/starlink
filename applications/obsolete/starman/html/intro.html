Stellar photometry programs

Starman's photometry programs allows the user to carry out the following tasks:-

Measuring

measure - Crowded field PSF photometry
automag - Automatic aperture photometry for bright isolated stars
simplemag - Isolated stars photometry
interact - Interactive measuring

Preparation and Analysis

starfind - Find star positions
profile - Determine the stellar profile for the measure program
chi - Reject stars with poor fits from measure program output
average - Average different photometry estimates (especially from measure )
interact - Inspect quality of measure program output
diagram - Plot CMDs and colour-colour diagrams

Other

addstars - Add fake stars from a table to images
dustring - Correct for dust rings
interact Interactive inspection of image
sprinkle - Make a table of field/cluster stars
unccd - Convert raw CCD frames to calibrated ones

Image handling programs -

Main interactive program

interact - Interactive image display

General programs

imcalc - Perform mathematical operations on image(s)
imcube - Cut a 2-D image from a 3-D or 4-D one
imcut - Cut an image out (sectioned,binned,inverted)
imdes - View:copy:write:delete the descriptors of an image
imfits_ dr - Read simple FITS image on disk
imflash - Simple display of image
imjoin - Join a number of images together
imkey - Insert:change image pixel values from keyboard
import - Load descriptors from other packages
imrotate - Rotate an image anti-clockwise by 90,180, or 270 degrees
imsmooth - Smooth an image
imstat - Get statistics of an image
imtype - Convert images between (real;integer;16-bit integer)
imweed - Set range of pixel values to Invalid

tbplot - Plot table graph/histogram - image picture/contour

Table handling programs

Main interactive program

tbsheet - Spreadsheet operations on a table

Calculator

tbcalc - Calculate table elements

Input, Output and Listing

tbdes - View:write:copy:delete the descriptors of a table
tbkey - Input numbers from keyboard into a table
tblist - Type and/or put in text file the contents of a table
tbload - Input a text file into a table
tbvalue - Type a table value
interact Inspect quality of measure program output

Joining

tbjoin - Join tables together

Graphical plot and Star chart

tbplot - Plot table graph/histogram - image picture/contour
tbchart - Plot out a realistic star map

General Programs

tbcomps - Find stars within/not within an annulus of each other
tbcut - Cut out parts of a table
tbmatch - Extract entries with matching names in two tables
tbnative - Convert table format from different machine machine style
tbnmatch - Put tables in the same order of names
tbpmatch - Extract entries with matching positions in two tables
tbrenum - Renumber names in a table
tbsort - Sort entries in a table
tbstat - Perform statistics on column(s) in tables(s)
tbtran_ auto - Transform one XY table to another automatically
tbtran_ do - Transform XY table positions using transform eqns
tbtran_ load - Calc transform eqn from a given magn, rotn, shift
tbtran_ make - Calc transform eqn from two matched XY tables
tbweed - Make a table with only those entries with a parameter in a range

Comparision with DAOPHOT and DoPHOT

Starman has distinct advantages compared with DAOPHOT and DoPHOT.

It can deal with very-crowded fields well, with images where the star diamters are less than a pixel, and where the star profile variea across the image.

For example. here is a user comment - " After 3 (find/measure/clean) runs Daophot still has undetected stars and quite a few stars are extremely dubious, especially when they start to appear in a grid like pattern. Starman however gets everything first time, and the cleaned images look quite nice!"

Push Buttons - The Panel

Many of these programs have the option of opening a panel of push-buttons, from which selections are made using point and click devices ( ie mice, trackballs and lightpens). A good example is the panel in interact for aperture photometry

By clicking again and again on the appropiate buttons different options can be done.
For example:-
click on image - a new image can be loaded into memory
click on flash - displays image, at the best contrast to see faint detail
click on cvalues - you can move the cursor over the image and see the pixel values

The Starman name

The name Starman is not an acronym - it comes from the title of the John Carpenter film. For information on the film, go to the Cardiff database , and search the titles for Starman. [The Cardiff database also has extensive information on many other films.]
