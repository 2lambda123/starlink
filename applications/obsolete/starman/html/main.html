Starman - Crowded-field stellar photometry

Starman is a software package for:-
  • Crowded-field stellar photometry
  • General stellar photometry
  • Image handling
  • Table handling.

Here is the way into the Starman hypertext help system.

The full User Guide, Starlink User Note 141, is in a Latex file. This is the front page .
The dvi version of SUN141 is also avaliable as the SUN/141 entry in the Starlink WWW page of Starlink User Notes. The Guide also exists in paper form at Starlink sites.

There is also a whats new page, and a newsgroup page. These are both kept at RAL.

The Fortran and C Source Code of the programs is also available.
