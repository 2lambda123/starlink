TBSORT -
Sort entries in a table

This takes a table and makes a new one with identical entries, but sorted on the values in a user defined column, either in ascending or descending order.

1.1 In

(Default - null)

The file containing the input table

The table must be a standard Starman table format file. It is a Starman ADAM .SDF image, with `real' 32-bit real values, with each column having a header of up to 20 characters, and each row having a name of up to 20 characters.

1.2 Numcol

(Default - 1
Range - 1 to last column)

The number of the column to do the sorting on.

1.3 Option

(Default - 'ascending'
Choice - 'ascending:descending')

Whether to sort into ascending or descending order.

1.4 Out

(Default - null)

The file to contain the output table.

The table will be a standard Starman table format file. It is a Starman ADAM .SDF image, with `real' 32-bit real values, with each column having a header of up to 20 characters, and each row having a name of up to 20 characters.

1.5 Title

(Default - input table title or 'Output from Tbsort'
Choice - any)

The Title to be added to the output file
