IMROTATE -
Rotate an image anti-clockwise by 90,180, or 270 degrees

Rotates an image anti-clockwise by 90, 180, or 270 degrees.

1.1 Degrees

(Default - '90'
Choice - `90:180:270')

Number of degrees anti-clockwise to rotate image.

1.2 In

(Default - null)

The name of the input image file.

The input image must be a standard Starman ADAM .SDF image, with either `real' 32-bit real values, or `short' 16-bit integer values.

1.3 Out

(Default - null)

The name of the new file containing the output image.

The output image will be a standard Starman ADAM .SDF image, with the same pixel types, either `real' 32-bit real values or `short' 16-bit integer values, as the input image.

1.4 Title

(Default - input image title, or if null, 'Output from IMROTATE' Choice - any)

Title to add to output image.
