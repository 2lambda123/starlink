IMCUBE -
Cut a 2-D image from a 3-D or 4-D image

This program cuts a 2-D image out of a 3-D or 4-D image.

It can squeeze down (add together) a range of planes in the other directions.

If in squeezing down, any of the 3-d or 4-d pixels, added in to make the output 2-d pixel, is INVALID, then the output pixel is INVALID.

The input image types it can deal with are:- 

     Type              Name
     ----              ----
     32-bit real       real
     32-bit integer    integer
     16-bit integer    short

1.1 In

(Default - null)

The name of the input image.

The input image must be a standard ADAM .SDF images, with either `real' 32-bit real values, `integer' 32-bit integer, or `short' 16-bit integer values.

1.2 Out

(Default - null)

The name of the output image.

The output image will be a standard ADAM .SDF images, with the pixel types of the input image, with either `real' 32-bit real values, `integer' 32-bit integer, or `short' 16-bit integer values.

1.3 Plane3

(Default - 'xy'
Choice - 'xy:xz:yz')

The plane to cut out of a 3-D cube. The cube is considered to have axes XYZ.

1.4 Plane4

(Default - 'xy'
Choice - 'xy:xz:xt:yz:yt:zt')

The plane to cut out of a 4-D cube. The cube is considered to have axes XYZT.

1.5 Title

(Default - input image title )

The title to be attached as a header to the output image.

1.6 Xplane

(Default - 1, X size
Range - 1 to X size of image)

The range, along the X direction, of planes to squeeze down (add together) to produce the plane to be cut out. A range of 1 (say 5,5) cuts out a single plane

For this to be asked for, the cube has to be either a 3-D one (XYZ), or a 4-D one (XYZT).

1.7 Yplane

(Default - 1, Y size
Range - 1 to Y size of image)

The range, along the Y direction, of planes to squeeze down (add together) to produce the plane to be cut out. A range of 1 (say 5,5) cuts out a single plane

For this to be asked for, the cube has to be either a 3-D one (XYZ), or a 4-D one (XYZT).

1.8 Zplane

(Default - 1, Z size
Range - 1 to Z size of image)

The range, along the Z direction, of planes to squeeze down (add together) to produce the plane to be cut out. A range of 1 (say 5,5) cuts out a single plane

For this to be asked for, the cube has to be either a 3-D one (XYZ), or a 4-D one (XYZT).

1.9 Tplane

(Default - 1, T size
Range - 1 to T size of image)

The range, along the T direction, of planes to squeeze down (add together) to produce the plane to be cut out. A range of 1 (say 5,5) cuts out a single plane

For this to be asked for, the cube has to be either a 3-D one (XYZ), or a 4-D one (XYZT).
