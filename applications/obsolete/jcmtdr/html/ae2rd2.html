JCMTDR - The Applications

AE2RD2

  • [JCMTDR main document]
This application performs the same function as AE2RD1 but rebins each input dataset individually using NAG interpolation routines (E01SAF and E01SBF) before coadding them into the result. Errors associated with the input datasets are not propagated.
