START_WAVE

Usage

START_WAVE=_REAL

Description

This parameter is only used when a user-specified wavelength scale is being generated (BIN_SIZE>0). It specifies the start wavelength for the rebinned wavelength scale.

The default is to calculate the start wavelength from the fitted wavelength scale when this parameter has value zero or is not specified.

INITIAL-SUGESTED-VALUE: 0.0.
