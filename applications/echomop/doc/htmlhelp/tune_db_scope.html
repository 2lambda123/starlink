TUNE_DB_SCOPE

Usage

TUNE_DB_SCOPE=_INTEGER

Description

This parameter sets the number of nearest-neighbours to take account of when constructing/using a reference feature database (FDB). It will not normally require adjusting.

The default value is 10.
