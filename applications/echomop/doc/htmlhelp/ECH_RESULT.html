ECH_RESULT

Function

Write results files.

Summary

ECHOMOP provides a set of different ways of generating results in the forms of graphs, ASCII listings of intensities, and Starlink standard sdf files which are ideal for further analysis.

This option controls the construction of sdf container files which may contain single orders, 2-D arrays containing all orders, or scrunched and merged whole frame spectra.

Spectra may be from the object or arc and may be either scrunched or as-extracted. In all cases the accompanying variance arrays and wavelengths are included in the container file. The module can also optionally list the stored values in ascii format to a .dmp ASCII file.
