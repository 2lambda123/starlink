TUNE_IDMDLT

Usage

TUNE_IDMDLT=_INTEGER

Description

Specifies the maximum number of neighbouring features (on each side) to examine when generating ratios for pattern matching.

Increasing the value of the parameter will lead to an exponential increase in CPU time. Only increase the value when all other possibilities for wavelength fitting have been exhausted.

The default value is 6.
