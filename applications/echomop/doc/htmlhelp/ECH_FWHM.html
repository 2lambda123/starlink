ECH_FWHM

Function

Estimates the full-width-half-max of ARC lines.

Summary

This routine cross correlates all local peaks in all orders to create an estimate of the profile of the reference lines. This profile is then fitted with a simple Gaussian, and the Full-Width-Half-Maximum (FWHM) calculated. The FWHM is used by the reference-line location and fitting routines. The average profile is plotted on the graphics device. A typical interactive plot appears below.
