TUNE_FFNYPLY

Usage

TUNE_FFNYPLY=_INTEGER

Description

Specifies the number of polynomial coefficients in Y for the flat-field model, and the re-fitted model.

The model of the flat field is made by fitting a polynomial at each X- and Y-increment. These polynomials are then used to calculate the balance factors at the 'Object' pixel postions.

The default value is 0.
