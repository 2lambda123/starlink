AUTO_ID

Usage

AUTO_ID=_LOGICAL

Description

Specifies that automation is required when arc line identification is being done.

Interactively, options are available to tune the fitted wavelength polynomial, add and delete features etc.

If AUTO_ID is TRUE then the package will try to identify the arc line features automatically. For orders with sufficient features (>12 lines) this will usually be successful.

A useful strategy is often to do an automatic run first, then manually examine the fitted features to verify their suitability.
