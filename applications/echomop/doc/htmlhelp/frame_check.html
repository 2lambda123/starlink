FRAME_CHECK

Usage

FRAME_CHECK=_LOGICAL

Description

Set FRAME_CHECK=TRUE if the processed frames are to be checked for bad row/columns.

The trace frame is used and the other frames are assumed to have identical bad rows/columns features.
