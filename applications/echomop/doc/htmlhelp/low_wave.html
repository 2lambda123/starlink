LOW_WAVE

Usage

LOW_WAVE=_REAL

Description

The shortest wavelength which you expect to be present in the spectrum.

The identification of lines proceeds faster for smaller search ranges, so specify a larger, rather than an arbitrarily small, value if you are guessing a value for LOW_WAVE.

The units in which the wavelength is expressed must be the same as those used in the .ARC line list file, normally Angstroms.
