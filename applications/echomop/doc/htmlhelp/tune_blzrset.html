TUNE_BLZRSET

Usage

TUNE_BLZRSET=_LOGICAL

Description

This parameter controls whether or not the blaze function is to be reset to unity after being applied.

The default is TRUE thus preventing accidental re-application of the blaze function to an extracted spectrum.

In some cases it may be necessary to keep the blaze function to apply it to a number of extracted object spectra. In this case set TUNE_BLZRSET=NO.

The default value is YES.
