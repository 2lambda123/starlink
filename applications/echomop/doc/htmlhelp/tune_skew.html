TUNE_SKEW

Usage

TUNE_SKEW=_REAL

Description

The number of bins the input array is to shifted (in mode 0 only). A positive value shifts data to HIGHER pixel numbers - this is a FIGARO convention, and is the opposite of that used by Lolita.
