TUNE_WVCNS

Usage

TUNE_WVCNS=_REAL

Description

Specifies the degree to which order wavelength scales may be inconsistent.

Expressed as a percentage of X-samples.

For example, if an order may be a maximum of 10% inconsistent as measured against the other orders, then TUNE_WVNCS should be set to 0.1.

The default value is 0.05.
