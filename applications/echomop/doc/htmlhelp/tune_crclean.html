TUNE_CRCLEAN

Usage

TUNE_CRCLEAN=_LOGICAL

Description

When set TRUE, optimal extraction will attempt to identify cosmic rays by examining the local variance.

This option will only effect optimal extraction, i.e., reductions with EXTRACT_MODE=O.

This option will not normally be used as more flexible tasks for cosmic-ray removal are provided separately to optimal extraction.

See also ECH_DECOSMIC.
