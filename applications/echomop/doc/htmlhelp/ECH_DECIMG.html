ECH_DECIMG

Function

Image cosmic ray pixels.

Summary

This option uses the quality array to determine which pixels have been flagged as contaminated by cosmic ray hits. It then takes the original object frame and makes a copy in which all the hit pixels are replaced by zero values. This frame could then be blinked with the original to visually assess the success of the cosmic ray locator.
