PHOTON_TO_ADU

Usage

PHOTON_TO_ADU=_REAL

Description

The number of incident photons corresponding to each count recorded by the detector.

Some detectors produce many electrons (counts) for each incident photon, or alternatively, one count for many photons. In order to be able to calculate using photon events it is necessary to take account of this factor. The PHOTON_TO_ADU parameter should be set appropriately.
