TUNE_SKYREJ

Usage

TUNE_SKYREJ=_INTEGER

Description

Specifies the number of reject cycles for which sky will be clipped, and the model re-fitted.

The polynomial model of the sky background is made by fitting a polynomial at each X-increment. These polynomials are then used to estimate the signal level at each spatial sample of the sky.
