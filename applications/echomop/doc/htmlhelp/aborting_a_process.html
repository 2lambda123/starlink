Aborting a Process

Previous topic | ECHOMOP HELP page | Starlink EDR home | Starlink Docs

Tasks may be aborted using /CTRL-C/, but this is not recommended as it may lead to corrupted files (in about 1% of cases).

A 'clean' task abort may be done by typing !! in response to any parameter prompt.

In echmenu, the recommended method of exit is to return to the main menu and then select the E/EXIT/99 option.
