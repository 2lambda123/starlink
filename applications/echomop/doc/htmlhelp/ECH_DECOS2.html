ECH_DECOS2

Function

Locate and flag cosmic-ray contaminated pixels.

Summary

This routine calculates the spatial order profiles and then compares each pixel to its predicted intensity. Ratios of observed/predicted intensities are calculated (per order) and outliers are clipped. This algorithm is iterated until a statistical test (K-S) determines that all the extraordinary pixels have been clipped. This is a very CPU intensive method which works best when the object spatial profile is well determined and broader than a few pixels.
