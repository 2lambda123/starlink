ECH_LINLOC

Function

Locate arc line candidates.

Summary

This routine locates (to the nearest X-pixel) the reference lines to be used for wavelength calibration purposes. Generally the reference spectrum will be an arc-lamp spectrum, but in principle this routine could more generally be used with a sky-line-rich sky spectrum, or even an object with strong emission lines (subject to provision of an applicable line list).

In the normal (arc lamp) case, this routine will be called once for each data frame. Usually there will be two arc lamp frames (bracketing the object exposure).
