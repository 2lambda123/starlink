Read-Only_RDCTN_file

The program is unable to obtain update access to an object in the reduction structure file.

This is probably because the file is protected against write access. Do an 

    % ls -alF -reduction-structure-file-.sdf

to check the file protection mask.

The program will continue but please note that it will be unable to write the results of the current step back into the reduction structure. You are therefore strongly recommended to copy the file to a location to which you have full access before proceeding.
