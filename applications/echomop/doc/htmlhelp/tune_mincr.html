TUNE_MINCR

Usage

TUNE_MINCR=_REAL

Description

Specifies the threshold level for deciding whether pixels located as 'possibly cosmic ray contaminated' should be flagged as such.

Only those pixels with intensities greater than TUNE_MINCR will be flagged. This provides a limit for the case where cosmic-ray identifiers become over enthusiastic due to the nature of the frame.

Normally this parameter should be set to zero, and therefore, have no effect.

INITIAL SUGGESTED VALUE: 0.0.
