Output Spectra

The reduced spectra may be output in various ways: 

    - A graphics plot file for printing.
    - An sdf file containing a single-order spectrum.
    - An sdf file containing one spectrum per order.
    - An sdf file containing a merged spectrum.
    - An ASCII file listing data pairs of wavelength vs flux.
