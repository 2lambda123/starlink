TUNE_TRCNS

Usage

TUNE_TRCNS=_REAL

Description

Specifies the degree to which traces may be inconsistent.

Expressed as a percentage of the number X-samples.

For example, for traces upto 10% inconsistent, as measured against the other traces, then set TUNE_TRCNS=0.1.

The default value is 0.05.
