TUNE_USEAAA

Usage

TUNE_USEAAA=_LOGICAL

Description

Set TRUE if the program is to make use of the `Astronomy & Astrophysics Abstracts' system of information categorisation.

This consists of a set of numbers corresponding to topics in Astronomy. If TUNE_USEAAA is active then the user will be shown the currently recognised categories and asked to select the one most relevant to the data being reduced. This parameter is only used in conjunction with the archiving facility, when TUNE_ARCHIVE=YES. The prompting process may be overridden by supplying a valid AAA code using the parameter TUNE_AAACODE = an integer.

INITIAL SUGGESTED VALUE: YES.
