TUNE_NOARC

Usage

TUNE_NOARC=_LOGICAL

Description

Set TRUE if there is no arc frame to use for wavelength calibration.

This may also be specified by replying NONE when prompted for the name of the arc frame (parameter ARC).

Default value: FALSE.
