No_trace_width

The program failed to determine the width of the order traces.

The algorithm for this process requires that the intensity falls to less than the fraction TUNE_TWTHR of the intensity at the peak.

The parameter may be adjusted (probably increasing it), or specified directly using the FIGARO command 

    SETOBJ object=-rdctn-file-.MORE.ECHELLE.TRACE_WIDTH value=nn

where nn is the approximate width in pixels.
