TUNE_MAXRFLN

Usage

TUNE_MAXRFLN=_INTEGER

Description

The maximum number of reference lines allowed for each order.

This parameter is used to dimension objects in the reduction structure file and should not normally need to be altered.

You may want to lower the value of this parameter to avoid the seeming profusion of 'false' line candidates produced by ECH_LINLOC, however, this is not a good idea if automatic line identification is being used as faint-feature positions are needed for the automatic ID to work well.
