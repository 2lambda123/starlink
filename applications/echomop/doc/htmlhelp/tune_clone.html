TUNE_CLONE

Usage

TUNE_CLONE=_CHAR

Description

The reduction structure file from which to clone information.

The reduction structure file should have been created by ECHOMOP previously. Objects which would normally be calculated by ECHOMOP will instead be copied from the named file. This allows the cloning of sections of a reduction amongst multi-frame datasets.
