TUNE_DIAGNOSE

Usage

TUNE_DIAGNOSE=_LOGICAL

Description

This parameter should be set TRUE only for short periods when a problem has occurred (and can be reproduced). It enables the production of a log file containing detailed information of use to the maintainer of ECHOMOP. This log is stored in the file

echomop_diagnostics.log

In the event of a problem mail this file, with details of the problem, to the Starlink software librarian at:

ussc@star.rl.ac.uk
