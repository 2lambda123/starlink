TUNE_INSTR

Usage

TUNE_INSTR=_CHAR

Description

The name of the instrument used to take the data. The package uses a definitions file with the name ECH_name_SPECIFIC.DEF to check for any useful information which may be present in the data frame headers.

To check which instruments have recognised headers use: 

    ls ADAM_PROG_* ECH*SPECIFIC.DEF.
