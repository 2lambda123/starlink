TUNE_OBJRTHR

Usage

TUNE_OBJRTHR=_REAL

Description

Specifies the reject-threshold for object pixels. Any pixels which deviate by more than TUNE_OBJRTHR sigma from the fitted polynomial model, will be clipped, and the model re-fitted. The polynomial model of the object_profile is made by fitting a polynomial at each x increment. These polynomials are then used to estimate the profile at the positions of the 'Object' pixels.

INITIAL SUGGESTED VALUE: 5.0.
