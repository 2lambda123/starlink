TUNE_FFNXREJ

Usage

TUNE_FFNXREJ=_INTEGER

Description

Specifies the number of reject cycles for which x will be clipped, and the model re-fitted. The polynomial model of the flat field is made by fitting a polynomial at each x and y increment. These polynomials are then used to calculate the balance factors at the positions of the 'Object' pixels.
