Cannot_create_ECH_RDCTN

The reduction structure file can not be created at this stage. You must either use ECHMENU Option 1 (ECH_CHECK_FRAME), or task ECH_LOCATE, as the first operation. These ensure that the reduction structure file used by all other task/operations is correctly created and initialised with the data frame dimension information.
