START_WAVE

Usage

START_WAVE=_REAL

Description

This parameter is only used when a user specified wavelength scale for the rebinned wavelength scale. The default is to calculate the start wavelength from the fitted wavelength scale if no value is supplied using this parameter.

INITIAL-SUGESTED-VALUE: 0.0.
