TUNE_MAXPOLY

Usage

TUNE_MAXPOLY=_INTEGER

Description

The maximum number of polynomial coefficients to be allowed for the duration of this reduction. The order of the polynomials used for each order may still be varied up to this limit, but may not be set to higher values. Also used for other types of fits such as SPLINE so a high value is recommended. For example, setting TUNE_MAXPOLY to 4 would permit the fitting of polynomials of the form: 

    constant + a.x + b.x-squared + c.x-cubed
