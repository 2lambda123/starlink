DISPLAY

Usage

DISPLAY=_LOGICAL

Description

Parameter is 'hidden'.

If DISPLAY=YES is used, then an imaging device is assumed to be available and to be displaying one of the frames under study. Some programs in the package will then overlay plots on the image to help illustrate their results. For example, the paths of the orders as fitted with polynomials.

INITIAL SUGGESTED VALUE: NO.
