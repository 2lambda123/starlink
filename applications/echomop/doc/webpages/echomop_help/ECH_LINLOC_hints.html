Hints

This task will normally report a LOT of line candidates. This is because it regards even very faint features as possible lines. The automatic line identification routine works best when it has access to a list of all features and the user is therefore encouraged NOT to tune ECH_LINLOC to reduce the number of candidates. The auto identifier starts by using only the brightest dozen lines so the result will not be biased by the faint features unless they are essential to the identification.
