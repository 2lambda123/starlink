TUNE_PREBAL

Usage

TUNE_PREBAL=_LOGICAL

Description

Set to YES if the flat field frame supplied has already been pre-processed by the user to contain balance factors for the pixel to pixel variations. In this case the modelling will not take place and the balance factors will simply be copied from the supplied frame. This should be used for special cases where the polynomial model cannot generate appropriate values.

NOTE: The removal of the blaze function is a separate operation and will require the use of the original raw flat field frame (or equivalent).
