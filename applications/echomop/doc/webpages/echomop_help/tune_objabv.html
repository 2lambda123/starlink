TUNE_OBJABV

Usage

TUNE_OBJABV=_INTEGER

Description

This parameter allows the extraction limit for the object to be set. A non-zero value for this parameter will over-ride the automatic object extent determination calculated by ECH_SPATIAL. It should only be used when the user is sure of the object extent and its position relative to the order traces. Typically it will be used when the user has extracted the first of a set of similar frames manually, and wishes to automate the extraction of the remaining frames.

INITIAL SUGGESTED VALUE: 0.
