New to ADAM

ADAM is a standard environment provided for Astronomical data processing on the UK STARLINK network systems. Prior to using ADAM programs, the environment must be setup.

To setup type 

    % source /star/etc/login

at the command line (dollar) prompt. You may place these commands in your login file to automate the setup (LOGIN.COM or .login).

ADAM tasks prompt the user for any information they require to operate. You may type ?/return/ at a programs' prompt and it will supply additional help about the information it is prompting for.

For extensive help on the facilities available you may normally type 

    HELP -package name

at the command line prompt.
