RESULT_TYPE

Usage

RESULT_TYPE=_CHAR

Description

This parameter selects the type of array which is to be saved into the output file. The following types are recognised: 

    - EXTOBJ, the unscrunched extracted object orders.


    - EXTARC, the unscrunched extracted arc orders.


    - SCROBJ, the scrunched extracted object orders.


    - SCRARC, the scrunched extracted arc orders.


    - OSPECT, the merged object spectrum.


    - CSPECT, the merged Std. Star calibration spectrum.

The default selection is EXTOBJ.
