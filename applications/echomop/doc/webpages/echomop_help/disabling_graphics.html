Disabling Graphics

ECHOMOP supports SOFT=NONE to disable all hard/soft graphics facilities which can save CPU time when such facilities are not required. This is not supported by other tasks at present and the syntax may change when STARLINK supports null graphics devices. The SOFT parameter should therefore be set using 

     SOFT=NONE or SOFT=NULL

to disable ECHOMOP graphics in the current release.
