TUNE_USE_NXF

Usage

TUNE_USE_NXF=_REAL

Description

Specifies the fraction of the x dimension of the frame to be used for determining the profiles. This is set low (0.1 = 10%) by default for speed. Very poor data may benefit from an increase in this parameter, providing a more accurate sampling of the object profile, and thus more accurate extraction weights. The central fraction from each order is sampled and the average profile over all orders is calculated. A special mode is selected by using the value TUNE_USE_NXF=1, which makes the sampling independent for each order.

The default values is 0.1. (i.e. 10%).
