TUNE_MXSKYPIX

Usage

TUNE_MXSKYPIX=_INTEGER

Description

Set to the maximum number of sky pixels allowed in the spatial direction for each order. This parameter is used to dimension objects in the reduction data structure and should only need to be altered when dealing with objects where the spatial extent is large. Note that increasing this parameter leads to a substantial increase in the size of reduction structure files, so it should be reset to smaller values when possible.
