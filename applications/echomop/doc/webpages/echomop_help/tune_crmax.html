TUNE_CRMAX

Usage

TUNE_CRMAX=_INTEGER

Description

The number of pixels expected to be contaminated by cosmic rays. This may be estimated by the user, or be generated by an automatic modeler. If no estimate is available, then zero should be used. The program will then display a histogram of the ratio of original/median image and allow manual selection of a clipping point. The number of pixels to be clipped (flagged as cosmic rays) will then be shown.
