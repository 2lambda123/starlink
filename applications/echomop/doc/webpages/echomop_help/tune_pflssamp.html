TUNE_PFLSSAMP

Usage

TUNE_PFLSSAMP=_INTEGER

Description

The maximum number of subsamples used when determining the subsampling rate to be used for the generation of the averaged profile. This parameter should only need to be changed when processing frames with a very large number of pixels sampling the object in the spatial direction.

The default value is 200 subsamples which will normally be more than is needed.
