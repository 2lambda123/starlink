TUNE_CLONE

Usage

TUNE_CLONE=_CHAR

Description

The name of a reduction structure file created with the package previously. The reduction objects which would normally be calculated by the task will be copied instead from the named reduction file. This makes it easy to clone sections of a reduction amongst multi-frame datasets.
