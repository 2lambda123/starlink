AUTO

Usage

AUTO=_LOGICAL

Description

Parameter is 'hidden'.

If AUTO is specified then ECHOMOP will use all the options available to automate the processes it invokes. Currently the following parameters will be affected: 

    - TRC_INTERACT     set to NO


    - TRC_VETO         set to NO


    - ID_INTERACTIVE   set to NO


    - CR_INTER         set to NO


    - 2D_INTERACT      set to NO


    - PFL_INTERACT     set to NO
