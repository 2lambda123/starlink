ARC

Usage

ARC=_CHAR

Description

Specifies an input structure.

The names of all reference images to be used for wavelength calibration must be supplied here. The number of images should be equivalent to the number specified for parameter NREF_FRAME. The image names should be separated by commas, for example: 

    reffrm1,reffrm2

Note that in most circumstances there will only be two reference frames. One taken prior to the object frame, and one subsequently. If you do not wish to provide an arc frame at all then you may reply NONE is response to this prompt or, alternatively set the hidden tuning parameter TUNE_NOARC = YES. If no arc is provided then the user must provide their own sets of wavelength feature positions, or wavelength scales, IF wavelength calibration is required.
