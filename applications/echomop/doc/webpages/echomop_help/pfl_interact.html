PFL_INTERACT

Usage

PFL_INTERACT=_LOGICAL

Description

When interactive profiling is selected the user can use a graphics cursor to specify the required attributes of each pixel in the spatial profile. The characteristics may be globally applicable, or specified independently for each order. In addition, any polynomial profiling fitting is also performed interactively, with the user able to adjust the degree of polynomials used and view the fits and residuals at each sub-sample position.
