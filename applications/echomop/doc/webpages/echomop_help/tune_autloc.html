TUNE_AUTLOC

Usage

TUNE_AUTLOC=_LOGICAL

Description

Set to YES if the positions of the orders are to be determined automatically. If FALSE then the user will be asked to select the order centres on a graphic display. This will generally only be useful when the user wishes to select a subset of the available orders in the frame, effectively ignoring the data in the remaining orders.

The default value is YES.
