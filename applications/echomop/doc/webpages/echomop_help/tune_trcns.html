TUNE_TRCNS

Usage

TUNE_TRCNS=_REAL

Description

Specifies the degree to which traces may be inconsistent. Expressed as a percentage of x samples. For example, if a trace may be a maximum of 10% inconsistent, measured against the other traces, then TUNE_TRNCS should be set to 0.1.

The default value is 0.05.
