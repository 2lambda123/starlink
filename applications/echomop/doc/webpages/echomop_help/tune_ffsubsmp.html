TUNE_FFSUBSMP

Usage

TUNE_FFSUBSMP=_LOGICAL

Description

Set to YES if subsampling is to be used when doing X-polynomial fits. This parameter is only used when polynomial fitting in the X-direction is enabled (TUNE_FFNXPLY > 1) and causes the program to subsample (10 samples per pixel) the spatial profile during fitting. This is useful when the user needs to calculate balance factors `outside' the dekker or at the very edge of the dekker where the intensity changes quickly from pixel to pixel.

INTIIAL SUGGESTED VALUE: NO.
