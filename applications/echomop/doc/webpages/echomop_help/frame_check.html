FRAME_CHECK

Usage

FRAME_CHECK=_LOGICAL

Description

Parameter is 'hidden'.

Set to TRUE if the processed frames are to be checked for bad row/columns (tracing frame is used and the other frames are assumed to have identical bad rows/columns.
