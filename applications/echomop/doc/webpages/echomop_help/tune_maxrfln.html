TUNE_MAXRFLN

Usage

TUNE_MAXRFLN=_INTEGER

Description

Set to the maximum number of reference lines per order which are allowed. This parameter is used to dimension objects in the reduction structure file and should not normally need to be altered. Some users may feel tempted to lower it to avoid the seeming profusion of 'false' line candidates produced by ECH_LINLOC. This is not a good idea if the automatic line identification mode is being used because it requires the presence of faint feature positions if it is to operate reliably.
