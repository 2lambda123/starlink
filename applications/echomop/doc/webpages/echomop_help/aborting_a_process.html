Aborting a Process

Previous topic | ECHOMOP HELP page | Starlink EDR home | Starlink Docs

The package tasks MAY be aborted using /CTRL-C/, but this is not recommended as it may lead to corrupted sdf files (approx 1% of cases). Task abort may be selected by typing !! in response to any parameter prompt. The recommended method of exit is to return to the ECHMENU main menu (if running) and then select the EXIT option.
