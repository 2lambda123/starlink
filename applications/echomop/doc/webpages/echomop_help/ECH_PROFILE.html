ECH_PROFILE

Function

Calculate a model for extraction profile.

Summary

The default is to create an average profile over all orders, but per-order (TUNE_USE_NXF=1), and polynomial modelling are also available for special cases (TUNE_OBJPOLY>0). A typical interactive plot appears below.
