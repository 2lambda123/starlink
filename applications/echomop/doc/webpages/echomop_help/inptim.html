INPTIM

Usage

INPTIM=_CHAR

Description

The name of the file holding the frame to be extracted. This will usually be the Object frame, however the Standard-Star frame may also be extracted. In cases where there are multiple frames of the same Object, then they may each be extracted by re-running this task and specifying a different frame each time (assuming the multiple data frames are perfectly registered).
