TRACIM

Usage

TRACIM=_CHAR

Description

The file name of frame to be used for the tracing of of the orders. Tracing may be attempted using a wide variety of data frame types. If the Object frame has good S/N then this should be used. In cases where the Object is faint, or exhibits large scale absorption features, then either the flat field OR Flux-calibration reference should be used.
