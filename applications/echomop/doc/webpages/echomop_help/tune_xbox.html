TUNE_XBOX

Usage

TUNE_XBOX=_INTEGER

Description

This parameter allows the sampling size (in x pixels) used when performing order location; to be altered. This should only be necessary for poor quality data where no other traceable frames can be used instead. Care must be taken that the sampling box is kept sufficiently small that all samples always come from the same order for each position of the sampling box. If in doubt use the IBOX function to overlay a box of appropriate size over the image concerned in order to check this is the case.
