TUNE_QUICK

Usage

TUNE_QUICK=_LOGICAL

Description

Set to YES if the frames are to be processed in Quick-look mode. This is designed to allow the fastest reduction by automatically tailoring parameters. In particular; all fits are clipped automatically, no flat fielding is performed, and a simple summed extraction is used. The object extent is assumed to have been specified by the user in parameters TUNE_OBJBLW and TUNE_OBJABV.

INITIAL SUGGESTED VALUE: NO.
