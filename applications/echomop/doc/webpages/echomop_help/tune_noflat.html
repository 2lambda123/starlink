TUNE_NOFLAT

Usage

TUNE_NOFLAT=_LOGICAL

Description

Set to YES if there is no flat field frame to use for balance factor calculation. This may also be specified by replying NONE when prompted for the name of the flat field frame. In the absence of a flat field frame, all balance factors are set to unity.

INITIAL SUGGESTED VALUE: NO.
