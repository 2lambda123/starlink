TUNE_SATRTN

Usage

TUNE_SATRTN=_REAL

Description

Specifies the threshold for deciding whether any pixels should be flagged as saturated. Such pixels will not contribute to the extraction in any way. Only pixels with intensities less than TUNE_SATRTN will be used by the subsequent operations.

INITIAL SUGGESTED VALUE: 1.0E20.
