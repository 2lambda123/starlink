FFIELD

Usage

FFIELD=_CHAR

Description

Specifies an input structure.

Specifies the name of the flat field image frame. The package uses this frame to calculate balance factors on a per-pixel basis. If the user would rather calculate their own balance frame using another package, then the balance frame should be supplied instead. In this case the hidden parameter TUNE_PREBAL should be set to YES to indicate that balance factors are being supplied. If you do not wish to supply a flat field at all then you may either reply NONE to this prompt, or alternatively set the hidden tuning parameter TUNE_NOFLAT=YES. If no flat field frame is supplied then the balance factors will be set to unity for all pixels.
