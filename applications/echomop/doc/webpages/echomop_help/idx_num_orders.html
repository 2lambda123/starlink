IDX_NUM_ORDERS

Usage

IDX_NUM_ORDERS=_INTEGER

Description

Specify which order is to be processed. IDX_ parameters provide a flexible method of indexing reduction structure arrays. Specifying a specific value causes processing of that order only. While specifying 0 causes an automatic loop through all values for the index (e.g. all orders).
