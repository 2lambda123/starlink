TUNE_FFLMED

Usage

TUNE_FFLMED=_LOGICAL

Description

Set to YES if the median is to be used when calculating the behaviour of the flat field using local pixels only. This applies only when TUNE_FFNXPLY=1 (no polynomial model), in which case local median/mean is taken over the nearest TUNE_FFLSMP pixels. This is useful for cases where there is significant small scale structure in the flat field.

INTIIAL SUGGESTED VALUE: NO.
