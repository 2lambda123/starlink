TUNE_BLZRSET

Usage

TUNE_BLZRSET=_LOGICAL

Description

This parameter controls whether or not the blaze function is to be reset to unity after being applied. The default is to do so, thus protecting the user from accidental re-application of the blaze function to an extracted spectrum. In some cases it may be required to keep the blaze function and apply it to a number of extracted object spectra. In this case the TUNE_BLZRSET=NO setting should be used.

The default value is YES.

INITIAL SUGGESTED VALUE: YES.
