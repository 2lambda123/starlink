TUNE_MERGE

Usage

TUNE_MERGE=_LOGICAL

Description

Set to YES if the scrunched object spectrum is to be merged into the 1D spectrum. The default is to zero the 1D spectrum before output. Setting TUNE_MERGE=YES allows the spectra from multiple frames to be co-added into the same output spectrum using a weighted merge.

The algorithm is the same as used in the FIGARO ECHMERGE program.

INITIAL SUGGESTED VALUE: NO.
