TRC_INTERACT

Usage

TRC_INTERACT=_LOGICAL

Description

Set to YES if the order fitting is to be done interactively. Automatic fitting is performed otherwise. When automatic fitting is engaged there are two tuning parameters which may be adjusted to optimise the performance of the program. These are TUNE_CLPMAXDEV which defines the clipping point, and TUNE_CLPBY which is the number of points to clip in each iteration.
