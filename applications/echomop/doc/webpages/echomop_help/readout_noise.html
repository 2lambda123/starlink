READOUT_NOISE

Usage

READOUT_NOISE=_REAL

Description

Some electronic detectors produce a significant amount of noise during the readout process. The older CCD's are often thus afflicted. If a non-zero value is supplied then it will be used to adjust the variances during extraction to take this error into account.
