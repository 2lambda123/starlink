ECH_RDUCD

Usage

ECH_RDUCD=_CHAR

Description

The name of the file to save the resulting spectrum into. The type of spectrum may vary from merely extracted, scrunched, 2D corrected etc. Only one spectrum per data file is saved, but this may be a 2D array of multiple orders.
