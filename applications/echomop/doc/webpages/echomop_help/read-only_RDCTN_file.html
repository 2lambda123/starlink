Read-Only_RDCTN_file

The program is unable to obtain UPDATE access to an object in the reduction structure file. This is probably due to the file being protected against write access. Do a 

    ls -alF -reduction-structure-file

to check the file protection mask.

The program will continue BUT please note that it will be unable to write the results of the current step back into the reduction structure. You are therefore strongly recommended to copy the file to a location where you have full access to it before proceeding.
