Arc Line Lists

Line list files are stored in ASCII format for ease of maintenance. ECHOMOP reads the line list(s) and creates its own proprietary file format. These feature databases are created by ECHOMOP itself using the ASCII format files as input. Line list filenames are expected to have a .ARC extension and will be looked for in the standard ADAM search path order.

The format of arc line lists is as follows 

    ! comment lines starting with a !


    nnnn.mmm   iii.jj    text


    nnnn.mmm   iii.jj    text


    nnnn.mmm   iii.jj    text

where 

    nnnn.mmm   is a wavelength in Angstroms
    iii.jj     is an (optional) intensity estimate
    text       is a description of the line e.g. Th II

In addition, a special comment line is recognised which allows the units of wavelength to be specified: 

    !WAVELENGTH_UNITS=some-unit=specifier

The default units are Angstroms and the package does not use the units string except for output messages. Future releases may support more intelligent use of the wavelength units in constraining fits etc.

User created lists should be placed in the directory pointed to by $ADAM_USER.
