TUNE_MINCR

Usage

TUNE_MINCR=_REAL

Description

Specifies the threshold for deciding whether any pixels located as 'possibly cosmic ray contaminated' should actually be flagged as such. Only those pixels with intensities greater than TUNE_MINCR will be flagged. This provides a limit for situations when cosmic ray identifiers become over enthusiastic due to the nature of the frame.

It will normally be set to zero and therefore have no effect.

INITIAL SUGGESTED VALUE: 0.0.
