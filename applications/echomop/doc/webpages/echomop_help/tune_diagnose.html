TUNE_DIAGNOSE

Usage

TUNE_DIAGNOSE=_LOGICAL

Description

This parameter should be set to YES for short periods only when a problem has occurred (and can be reproduced). It enables the production of a log file containing detailed information of use to the maintainer of the package. This log is stored in ECHOMOP_DIAGNOSTICS.LOG and the command @ECHOMOP_SOURCE:ECH_BUGSWAT can be used to automatically MAIL the latest version to the maintainer. The user should of course also send a message detailing the circumstances under which the problem is observed.
