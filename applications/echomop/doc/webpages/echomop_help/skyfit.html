SKYFIT

Usage

SKYFIT=_CHAR

Description

This parameter allows the selection of various different functions to be used for a fitting procedure. The recognised functions are MEAN(the default),POLY and SPLINE. NOTE that the maximum number of coefficients will differ for the functions. For POLY the maximum number of coeffs corresponds to the number of degrees+1, so for a cubic use 4 coeffs. For a SPLINE the maximum number of coeffs corresponds to (knots+7)*2, so for 3 knots use 20 coeffs. If you are using spline fits at all then you should also ensure that the value of TUNE_MAXPOLY is set accordingly, as well as using appropriate values for TUNE_SKYPOLY and/or TUNE_SKYXPLY.

INITIAL SUGGESTED VALUE: MEAN.
