TUNE_DEKTHR

Usage

TUNE_DEKTHR=_REAL

Description

The threshold for locating the dekker limits is the percentage (of maximum intensity) threshold below which the intensity is to fall before being classified as outside the dekker. In general an ARC frame will be used to establish the dekker limits, and a value of 80% will suffice.

The default value is 0.8. (i.e. 80%).
