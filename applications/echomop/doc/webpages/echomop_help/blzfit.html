BLZFIT

Usage

BLZFIT=_CHAR

Description

This parameter allows the selection of various different functions to be used for a fitting procedure. The recognised functions are POLY (the default) and SPLINE. NOTE that the maximum number of coefficients will differ for each case. For POLY the maximum number of coeffs corresponds to the number of degrees+1, so for a cubic use coeffs=4. For a SPLINE the maximum number of coeffs corresponds to (knots+7)*2, so for 3 knots use coeffs=20. Thus if you are using spline fits at all then you should ensure that the value of TUNE_MAXPOLY is set accordingly, as well as using an appropriate value for BLZ_NPOLY.

INITIAL SUGGESTED VALUE: POLY.
