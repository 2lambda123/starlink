ID_INTERACTIVE

Usage

ID_INTERACTIVE=_LOGICAL

Description

Specifies that interaction is required when arc line identification is being done. Many options are available in this mode to allow the user to tune the fitted wavelength polynomial, add/delete features etc. If ID_INTERACTIVE is set to NO then the package will try to identify the arc line features automatically. For orders with sufficient features this will usually be successful (>12 lines), and a useful strategy is probably to do an automatic run first, and then manually examine the fitted features to verify their suitability.
