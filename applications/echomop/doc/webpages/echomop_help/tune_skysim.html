TUNE_SKYSIM

Usage

TUNE_SKYSIM=_LOGICAL

Description

Set to YES if simulations are to be used to estimate the variance of the sky using polynomial fits in the wavelength direction. These simulations are CPU intensive.

The default value is NO.
