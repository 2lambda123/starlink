BLZ_NPOLY

Usage

BLZ_NPOLY=_INTEGER

Description

The default number of coefficients to be used when fitting functions to the order blaze The type of function to be used is set using the parameter BLZFIT (default is POLY for polynomial). When BLZFIT=POLY then the number of coeffs should be set to degree of polynomial + 1. When BLZFIT=SPLINE the number of coeffs should be set to (number of knots + 7) * 2. For example, setting BLZ_NPOLY to 4 would permit the fitting of polynomials of the form: 

    constant + a.x + b.x-squared + c.x-cubed

INITIAL SUGGESTED VALUE: 7.
