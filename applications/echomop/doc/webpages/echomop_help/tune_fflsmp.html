TUNE_FFLSMP

Usage

TUNE_FFLSMP=_INTEGER

Description

Specifies the number of samples per local mean/median to use when the flat field model is based on the local behaviour only. This parameter is only used when the x polynomial degree (TUNE_FFNXPLY) is set to 1 (would imply constant flat field). The parameter TUNE_FFLMED controls whether a local mean or median is used.

INITIAL SUGGESTED VALUE: 10.
