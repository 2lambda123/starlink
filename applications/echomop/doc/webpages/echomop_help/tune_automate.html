TUNE_AUTOMATE

Usage

TUNE_AUTOMATE=_CHAR

Description

This hidden parameter allows multiple options to be automatically invoked upon startup by the monolithic processing task. Any parameters required must have been supplied on the command line or the user will be prompted as usual. For example: 

    TUNE_AUTOMATE=1,2,3,EXIT

would run the first 3 processing options and then exit.
