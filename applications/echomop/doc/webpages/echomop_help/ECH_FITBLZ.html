ECH_FITBLZ

Summary

This option consists of two steps as follows: 

    11.1  Model blaze function using a polynomial


    11.2  Apply blaze function to extracted spectrum

The polynomial fit may be controlled by the user using a graphics plot and cursor if required. The polynomial is scaled so as to conserve the median intensity of the spectrum. A typical interactive plot appears below.

Once a blaze polynomial has been applied to an order spectrum it is automatically reset to unity. This is done to ensure that it cannot be re-applied to the same order spectrum by mistake.
