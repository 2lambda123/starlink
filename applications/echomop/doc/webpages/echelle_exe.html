ECHOMOP version 3.3-0 pre-release

This page contains links to the available test versions of ECHOMOP and to copies of the new arc line database files THAR.sdf and THAR.ARC.

This build is likely to be the one included on the forthcoming Starlink CD. See the notes on changes for this version.

Please note that, there may well be changes made to these before they are made available in the Starlink Software Collection.

New arc line databases

This is a minor change for anyone interested. The revised databases contain some lines missing in earlier versions; the lines have been added from the Figaro line lists.

The Thorium-Neon database was supplied by Michael Albrow.

  • THAR.sdf.gz Thorium-Argon database.
  • THAR.ARC.gz Thorium-Argon line list.
  • THNE.sdf.gz Thorium-Neon database.
  • THNE.ARC.gz Thorium-Neon line list.
  • CUAR.sdf.gz Copper-Argon database.
  • CUAR.ARC.gz Copper-Argon line list.

Pre-release ECHOMOP versions available

  • echomop_alpha_OSF1_alpha.tar.Z - Digital UNIX version.
  • echomop_sun4_Solaris_alpha.tar.Z - Solaris version.
  • echomop_ix86_Linux_alpha.tar.Z - Linux version.

    Instructions for setting up the test versions

    • Copy the .Z file to an empty directory where you will "install" the test version. You need about 9 Mbytes for the completed installation, up to 17 Mbytes are needed during the installation procedure.

    • Uncompress and extract the archive, e.g.: 
         % zcat echomop_*_alpha.Z | tar xvf -
    • To make the installation: 
         % make alphainstall

    Using the test version

    You will be able to setup for using the test version of ECHOMOP by "source"ing the file echalpha.csh in the installation directory.

    NOTE: you have to run the standard Starlink "echomopsetup" prior to the test-version set up as the test version still uses datafiles from the standard installation.

    Once you have the test version set up you can use "echomop" to get back to the Starlink release and "echalpha" to get to the test version.
