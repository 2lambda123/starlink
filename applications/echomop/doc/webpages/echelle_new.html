Most recent changes

  • The third alpha test version of ECHOMOP version 3.2-1 has been made available.

New before 18th April 1997

  • SUN/152 and SG/9 have been updated.
  • The version number for the next ECHOMOP release has been changed from 3.1-1 to 3.2-0. This reflects the large number of fixes and small changes made. The are also a few significant changes, hopefully improvements, to the user interface.

New before 15th July 1996

  • The Echelle Data Reduction Cookbook has been updated.
  • Access to my test builds of ECHOMOP is now available.
  • The Introduction to Echelle Spectroscopy has been corrected and updated.
  • An update on the current status of ECHOMOP has been added. This includes a list of fixes made to the current release of ECHOMOP.

New before 29th April 1996

  • An index has been added.
  • The Introduction to Echelle Spectroscopy has moved.
  • The Echelle Data Reduction Cookbook has moved.
  • There is a working bug report page.
  • A new draft version of the Echelle Data Reduction Cookbook (SC/3) is available.
  • A draft version of the Introduction to Echelle Spectroscopy (SG/9) is available.

New before 22nd January 1996

  • A draft version of the Echelle Data Reduction Cookbook (SC/3) is available.
  • A draft version of the ECHOMOP documentation in hypertext form is available.
  • The ECHOMOP HELP text is available in a "draft" hypertext form.

New before 3rd November 1995

  • The system was set up - everthing is new!
